## Структуры и работа с файлами в C++

### Условие задачи (Вариант 18)

Определить структурный тип, описывающий компьютерные игры (название игры, студия разработчик, стоимость подписки, объем аудитории игроков, поддерживаемые платформы: PC, mobile, gamepad; жанры). Заполнить структурный массив 20-ю записями.

Необходимо выполнить следующие действия:

1. Переписать из исходного массива в другой массив информацию только об авиасимуляторах с аудиторией от 1 миллиона пользователей.
2. Новый массив отсортировать по поддерживаемым платформам.
3. Вывести все данные по конкретной игре (по названию).
4. Вывести 3 самых дорогих игр.
5. Реализовать функцию изменения данных об игре.
6. В отдельный массив поместить все стратегии для PC.
7. Реализовать вывод отфильтрованных данных в виде оберточной функции.

### Работа с файлами (Занятие 10)

1. Чтение из текстового файла `costs.txt` — обновление стоимости игр по названию.
2. Чтение и запись структуры в бинарный файл `games.dat` — оформлено в виде подпрограмм `readBinaryGames` и `writeBinaryGames`.

### Особенности реализации

- Каждая подзадача вынесена в отдельную функцию.
- Присутствует поле перечисляемого типа `enum Platform`.
- В качестве одного из полей структуры используется вложенная структура `GenreSet`.
- Данные инициализируются внутри программы в функции `init()`.
- Сортировка выполнена алгоритмом пузырька с передачей функции сравнения.

### Структура проекта

- `main.cpp` — исходный код программы.
- `games.dat` — бинарный файл с данными (создаётся при запуске).
- `costs.txt` — текстовый файл для обновления стоимости (создаётся при запуске).

### Требования к сборке

- Компилятор с поддержкой C++17.
- Стандартная библиотека fstream, cstring.
